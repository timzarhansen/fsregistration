# Pretrained Weights for 3D Registration Models

Place pretrained model weights in the appropriate subdirectory.

## GeoTransformer
- **File**: `geotransformer/geotransformer-3dmatch.pth`
- **Download**: https://github.com/qinzheng93/GeoTransformer/releases
- **Dataset**: 3DMatch

## HybridPoint
- **File**: `hybridpoint/hybridpoint-3dmatch.pth`
- **Download**: `https://github.com/liyih/HybridPoint/blob/main/weights_for_hybrid/3dmatch.tar`
- **Dataset**: 3DMatch

## RegTR
- **File**: `regtr/regtr-3dmatch.pth`
- **Download**: https://github.com/yewzijian/RegTR/releases
- **Dataset**: 3DMatch

## Predator (KPFCNN)
- **File**: `predator/predator-indoor.pth`
- **Note**: KPFCNN model pretrained on indoor datasets
- **Dataset**: Indoor
